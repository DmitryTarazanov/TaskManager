Добрый День! Для проекта использовал БД PosrgreSQL.  
В классе Constans находятся константы, для подключения к Базе данных. Необходимо вписать акутальные значения. 
В классе JsonFileParser находится объект с именем файла, для чтения из него. В переменную fileRoot необходимо
прописать путь до файла с данными. 
Приложение само при запуске создаст таблицу в БД, создавать ее заранее нет необходимости. В папке так же есть файл 
SomeData с тестовыми данными.
